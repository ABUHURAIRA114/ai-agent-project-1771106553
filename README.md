# Console Calculator Project
## Overview
A simple console-based calculator implemented in Python.

## Features
* Basic arithmetic operations: addition, subtraction, multiplication, division
* Supports multiple operands and operators

## Requirements
* Python 3.8 or higher

## Usage
1. Clone the repository: `git clone https://github.com/your-username/calculator.git`
2. Navigate to the project directory: `cd calculator`
3. Run the calculator: `python calculator.py`

## Code Structure
* `calculator.py`: Main calculator script
* `operations.py`: Module containing arithmetic operation functions
